# 概述

**BedrockPlayerSupport** 是一款专为 Paper 端打造的实用辅助插件，旨在全面提升 Minecraft Java 版服务器中**基岩版玩家**的游玩体验

它通过为服务器的各种常用命令创建基于 Form（表单）的交互界面，解决了基岩版玩家（尤其是手机触屏和手柄玩家）输入长串指令不便的痛点，让交互变得前所未有的直观和顺滑

## ✨ 核心特性

* **全表单化操作**：将繁琐的文本指令转化为可视化的点击菜单
* **多功能覆盖**：原生支持并重构了玩家常用的传送、社交与福利等功能界面，包含：
  * **家系统**
  * **玩家传送**
  * **死亡点返回**
  * **私聊系统**
  * **礼包系统**
* **基岩版自动注册登录**：无缝对接登录插件，进服自动完成注册与登录验证，无需手动输入繁琐指令

## 🛠️ 运行环境

为了确保插件正常运行，你的服务器需要满足以下条件
* **服务端核心**：Paper 或其分支 (Purpur, Folia 等)
* **前置插件**：必须安装 [Geyser](https://geysermc.org/wiki/geyser/) 与 [Floodgate](https://geysermc.org/wiki/floodgate/) 才能向基岩版玩家正确发送表单
