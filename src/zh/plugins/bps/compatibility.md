# 兼容性

**BedrockPlayerSupport** 的功能分为两部分：一部分需要特定的基础核心插件支持，另一部分则是独立运行或具有通用兼容性的功能

## 1. 基础插件依赖功能

以下功能需要服务器安装并运行对应的主流基础插件（五选一即可）

| 功能模块                        |  CMI  | EssentialsX | HuskHomes | AdvancedTeleport | SunLight |
| :------------------------------ | :---: | :---------: | :-------: | :--------------: | :------: |
| **接收传送请求** (TPA Accept)   |   ✅   |      ✅      |     ✅     |        ✅         |    ✅     |
| **发起传送表单** (TPA / TP)     |   ✅   |      ✅      |     ✅     |        ✅         |    ✅     |
| **个人家园系统** (Home)         |   ✅   |      ✅      |     ✅     |        ✅         |    ✅     |
| **公共家园/地标** (Public Home) |   ❌   |      ❌      |     ✅     |        ❌         |    ❌     |
| **返回上一点/死亡点** (Back)    |   ✅   |      ✅      |     ✅     |        ✅         |    ✅     |
| **服务器传送点** (Warp)         |   ✅   |      ✅      |     ✅     |        ✅         |    ✅     |
| **领取礼包系统** (Kit)          |   ✅   |      ✅      |     ❌     |        ❌         |    ✅     |

## 2. 独立与通用功能

以下功能不依赖上述基础插件，或具有极广泛的兼容性，几乎可以在任何服务器环境中使用

| 功能模块               | 兼容性说明   | 依赖要求                                                                                        |
| :--------------------- | :----------- | :---------------------------------------------------------------------------------------------- |
| **私聊系统** (Message) | **完全独立** | **无需任何基础插件**，原版服务端已有私聊指令，可直接为基岩版玩家提供表单对话界面                |
| **自动注册登录**       | **通用支持** | **理论支持所有登录插件**（如 AuthMe, FastLogin 等）通过无缝对接验证协议，实现进服即自动完成验证 |
