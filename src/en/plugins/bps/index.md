# Overview

**BedrockPlayerSupport** is a highly practical utility plugin built specifically for Paper servers, designed to comprehensively enhance the gameplay experience of **Bedrock Edition players** on Minecraft Java Edition servers.

By creating Form-based interactive GUIs for various common server commands, it solves the pain point of typing long commands for Bedrock players (especially those using touch screens and controllers), making interactions smoother and more intuitive than ever before.

## ✨ Core Features

* **Fully Form-Based Operations**: Converts tedious text commands into visual, clickable menus.
* **Multi-Function Coverage**: Natively supports and overhauls commonly used interfaces for teleportation, social features, and rewards, including:
  * **Home System**
  * **Player Teleportation (TPA/TP)**
  * **Return to Death Point (Back)**
  * **Private Messaging**
  * **Kit System**
* **Auto Login/Register for Bedrock**: Seamlessly integrates with authentication plugins to automatically complete registration and login upon joining, eliminating the need to manually type tedious login commands.

## 🛠️ Requirements

To ensure the plugin runs correctly, your server must meet the following conditions:
* **Server Software**: Paper or its forks (Purpur, Folia, etc.)
* **Dependencies**: [Geyser](https://geysermc.org/wiki/geyser/) and [Floodgate](https://geysermc.org/wiki/floodgate/) must be installed to correctly send forms to Bedrock players.
